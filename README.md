# ScienceIO Example Streamlit Search App

Hi! Welcome to our example search engine repo. With this, you can understand how to set up a Streamlit app. 

To run the app, first clone this repo, then run the following commands:

```
pip install -r requirements.txt
streamlit run app.py

```

## Data
Data for this example app was downloaded from [LitCovid](https://www.ncbi.nlm.nih.gov/research/coronavirus/#data-download) on October 21st, 2022.
